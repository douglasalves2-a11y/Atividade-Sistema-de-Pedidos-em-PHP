# Sistema de Gestão de Pedidos em PHP

## 📌 Descrição
Sistema simples desenvolvido em PHP para gerenciar pedidos de uma loja.

## 🧱 Estrutura
- Cliente
- Produto
- Pedido
- ItemPedido

## 🔗 Relacionamentos
- Associação: Pedido → Cliente
- Agregação: Pedido → ItemPedido
- Composição: ItemPedido depende do Pedido
- Dependência: métodos que recebem objetos

## 🚀 Como executar
1. Baixar o projeto
2. Executar o arquivo index.php
